Instructions to Push Changes to GitHub

Step 1: Open Command Prompt or PowerShell

Open a command prompt or PowerShell window in your project directory:

E:\Full- Stack -Django-Project\sports-league-organizer

Step 2: Add Changes

Add all modified files to the staging area:

git add .

Step 3: Commit Changes

Commit the changes with a descriptive message:

git commit -m "Fix accessibility issues and implement team logo upload functionality"

Step 4: Push to GitHub

Push the changes to your GitHub repository:

git push origin master

If You Encounter Issues

If you encounter any issues with pushing, try the following:

  1. If there are conflicts, pull the changes first and then push:

    git pull --rebase origin master
git push origin master

  2. If you're prompted for credentials, enter your GitHub username and personal access token (not your password).

  3. If you need to force push (use with caution):

    git push -f origin master
